🍕 Pizza Sales Report
📌 Project Overview

This project focuses on tracking and analyzing pizza sales data for a specific year. The goal is to extract meaningful insights from raw sales data and present them through an interactive and visually appealing dashboard.

By leveraging data analytics and visualization techniques, this project helps in understanding sales trends, customer preferences, and overall business performance.

🎯 Objectives
Analyze yearly pizza sales data
Identify top-performing pizzas and categories
Track revenue, orders, and sales trends over time
Provide actionable insights for better business decisions
🛠️ Technologies Used
Power BI – For data visualization and dashboard creation
SQL Server Management Studio (SSMS) – For data extraction, querying, and preprocessing
📊 Key Features
Interactive dashboard with filters and slicers
KPI metrics such as total revenue, total orders, and average order value
Sales breakdown by:
Category
Size
Time (daily, monthly trends)
Top and bottom performing products analysis
📁 Project Structure
Dataset – Contains raw pizza sales data
SQL Queries – Data extraction and transformation scripts
Power BI Report – Dashboard file (.pbix)
Final Output – Exported dashboard in PDF format
